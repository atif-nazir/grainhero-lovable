🎯 STEP-BY-STEP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Open: QUICK_START.md
   └─ Get the 5 values you need

2. Add to Vercel:
   └─ Settings → Environment Variables → Add 5 variables

3. Run SQL:
   └─ Copy scripts/001_init_schema.sql
   └─ Paste in Supabase SQL Editor
   └─ Click Run

4. Create Webhook:
   └─ Stripe → Developers → Webhooks → Add endpoint
   └─ URL: https://your-domain.com/api/webhooks/stripe
   └─ Copy signing secret → Add to Vercel as STRIPE_WEBHOOK_SECRET

5. Test:
   └─ npm run dev
   └─ Visit /auth/sign-up
   └─ Verify everything works

6. Deploy:
   └─ Push to GitHub
   └─ Vercel auto-deploys
   └─ You're live! 🎉
